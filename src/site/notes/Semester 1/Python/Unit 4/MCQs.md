## Classes & Objects

### Question 1
What is a class in Python?

*   B) A function
*   C) A module
*   D) A variable

<details>
<summary><strong>Check Answer</strong></summary>

> **Correct Answer: A) A blueprint for creating objects**
</details>

### Question 2

*   A) class
*   B) def
*   C) object
*   D) create

<details>
<summary><strong>Check Answer</strong></summary>

## Inheritance

### Question 1
What is inheritance in OOP?
*   A) A class inheriting properties from another class
*   B) A function calling another function
*   C) A variable assignment
*   D) None of the above

<details>
<summary><strong>Check Answer</strong></summary>

</details>
