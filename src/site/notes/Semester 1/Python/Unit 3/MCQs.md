## Recursion

### Question 1
What is recursion?
*   A) A function that calls itself
*   B) A function that calls another function
*   C) A loop structure
*   D) None of the above

## Modules

### Question 1
Which keyword is used to import a module in Python?
*   A) include
*   B) import
*   C) require
*   D) use

<details>
<summary><strong>Check Answer</strong></summary>

> **Correct Answer: B) import**
</details>

### Question 2
How do you import only specific functions from a module?
*   A) import module.function
*   B) from module import function
*   C) include function from module
*   D) require module.function

<details>
<summary><strong>Check Answer</strong></summary>

> **Correct Answer: B) from module import function**
</details>

## Exception Handling

### Question 1
Which keyword is used to handle exceptions in Python?
*   A) catch
*   B) except
*   C) handle
*   D) error

<details>
<summary><strong>Check Answer</strong></summary>

> **Correct Answer: B) except**
</details>

### Question 2
What is the purpose of the finally block?
*   A) To handle exceptions
*   B) To execute code regardless of exceptions
*   C) To raise exceptions
*   D) None of the above

<details>
<summary><strong>Check Answer</strong></summary>

> **Correct Answer: B) To execute code regardless of exceptions**
> 
> *`finally` block always executes, whether exception occurs or not.*
</details>
